# Python on GitHub Pages 🐍

This project demonstrates how to **run Python code in the browser** using **Pyodide** on [GitHub Pages](https://pages.github.com/).  

> ⚠️ Note: GitHub Pages cannot run `.py` files directly. This project uses Pyodide to execute Python code in the browser.

---

## How It Works

1. `index.html` loads the **Pyodide library**.
2. Your Python code is stored in a JavaScript variable (`pythonCode`).
3. Clicking the **Run Python** button executes the Python code and shows the output.

---

## Demo

- Open the live site:  
`https://yourusername.github.io/python-copy/`

- Click **Run Python** to see the output from the Python script.
